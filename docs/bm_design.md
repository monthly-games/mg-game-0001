## 3. 상세 BM 설계

### 3.1 캐릭터/영웅 수집 BM

#### 무료 구조
- 스토리/튜토리얼 완료 시 기본 영웅 5종 지급
- 일일 미션, 업적으로 뽑기권 획득
- 노멀/레어 급 15종 이상 무료 파밍 가능

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 일반 뽑기 | 10연차 1회 | 프리미엄 재화 3,000 |
| 픽업 배너 | 신규/한정 영웅 확률 UP | 동일 |
| 천장 시스템 | 80연차 최고 등급 확정 | - |
| 선택권 패키지 | 10연차 + SR 선택권 | $9.99 |
| 시즌 버전 | 동일 영웅 한정 스킨 버전 | 개별 판매 |

### 3.2 성장/강화 BM

#### 무료 구조
- 방치 전투 + 일일 퀘스트로 레벨업 재화 획득
- 주간 보스에서 고급 강화 재료 드롭
- 전생/리셋 무료 1회/주

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 스타터 패키지 | 레벨업 재화 + 장비 세트 | $4.99 |
| 성장 부스트 패스 | 경험치/골드 +50% (30일) | $9.99/월 |
| 전생 편의 패키지 | 리셋 횟수 증가, 손실 감소 | $2.99 |

### 3.3 퍼즐/스테이지 BM

#### 무료 구조
- 스태미너 최대 60, 5분당 1 회복
- 일일 무료 충전 1회
- 챕터 클리어 보상으로 재화 충분 지급

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 스태미너 즉시 충전 | 60 회복 | 프리미엄 재화 100 |
| 스태미너 패스 | 최대치 +30, 회복 속도 2배 | $4.99/월 |
| 스킵 티켓 | 클리어한 스테이지 자동 완료 | 이벤트/상점 |
| 하드 모드 입장권 | 고급 보상 던전 입장 | 프리미엄 재화 50 |

## 6. BM 밸런스 원칙

### P2W 방지선
- 최고 등급 영웅 없이도 메인 스토리 100% 클리어 가능
- PVP 상위 30%는 무과금 도달 가능
- 과금 우위는 **시간 단축 + 수집 속도**에 한정

### 과금 압박 최소화
- 일일 플레이만으로 핵심 콘텐츠 모두 접근 가능
- 스태미너 부족으로 플레이 불가 상황 최소화
- 천장 시스템으로 뽑기 좌절 방지

## 1. BM 개요

|------|------|
| 게임 타이틀 | Simple Tower Defense |
| 장르 | Tower Defense, Strategy |
| 가격 모델 | F2P + IAP + 리워드 광고 |
| 타이틀 성격 | 중단기 라이브 (캐주얼 전략 게임) |
| 주요 수익원 | 리워드 광고, 타워 스킨, 편의 기능, 골드 패키지 |

**핵심 철학**: **P2W 최소화**, 광고와 편의/장식 IAP 중심의 건강한 수익 모델

## 2. 수익 구조 요약

```
┌─────────────────────────────────────────────────────────┐
│            타워 디펜스 수익 구조 비중                     │
├─────────────────────────────────────────────────────────┤
│  리워드 광고 (Rewarded Ads)       50%                    │
│  타워 스킨/장식 IAP               20%                    │
│  편의 기능 (골드 부스트 등)       15%                    │
│  광고 제거 패스                   10%                    │
│  스페셜 타워 IAP                   5%                    │
└─────────────────────────────────────────────────────────┘
```

**설계 원칙**:
- 광고 중심 F2P 친화적 설계
- 성능 격차는 ±20% 이내 (스페셜 타워도 밸런스 유지)
- 스킨/장식은 성능 영향 없음 (비주얼 only)
- 편의 기능은 시간 단축/품질 개선 목적

## 3. 상세 BM 설계

### 3.1 리워드 광고 BM (50% 수익 기여)

타워 디펜스의 핵심 수익원. 무료 유저도 충분히 즐길 수 있게 하되, 광고로 더 빠른 진행 지원.

#### 광고 배치 포인트

| 광고 위치 | 트리거 조건 | 보상 | 일일 제한 |
|----------|-----------|------|----------|
| **웨이브 클리어 보상 2배** | 웨이브 완료 후 선택 | 골드 획득량 2배 | 10회 |
| **골드 부스트** | 메인 UI에서 선택 | 30분간 골드 +50% | 3회 |
| **부활 (Life 회복)** | Life가 5 이하일 때 | Life +5 회복 | 5회 |
| **무료 타워 업그레이드** | 타워 선택 시 | 1개 타워 즉시 업그레이드 | 3회 |
| **웨이브 스킵** | 웨이브 시작 전 선택 | 해당 웨이브 자동 완료 | 3회 |
| **일일 보너스 골드** | 메인 메뉴에서 선택 | 500 골드 지급 | 1회 |

#### 광고 설계 원칙
- **강제 광고 없음**: 모든 광고는 선택형 (Rewarded only)
- **쿨다운**: 각 광고 타입별 30초 쿨다운 (스팸 방지)
- **일일 캡**: 총 25회/일 (과도한 광고 시청 방지)
- **첫 3분 차단**: 게임 시작 후 3분간 광고 배치 금지

#### 예상 효과
- ARPDAU: $0.05~0.08 (광고 기반 게임 평균)
- 광고 시청률: 40~60% (유저의 절반 이상이 광고 활용)

---

### 3.2 타워 스킨/장식 IAP (20% 수익 기여)

**핵심**: 성능에 영향 주지 않는 순수 비주얼 커스터마이징

#### 스킨 유형

| 스킨 타입 | 설명 | 가격대 | 예시 |
|----------|------|--------|------|
| **타워 스킨** | 타워 외형 변경 (성능 동일) | $1.99~3.99 | 사이버펑크 타워, 판타지 타워 |
| **발사체 이펙트** | 총알/미사일 이펙트 변경 | $0.99~1.99 | 레이저 빔, 불꽃 화살 |
| **승리 이펙트** | 웨이브 클리어 시 특수 연출 | $0.99 | 폭죽, 별똥별 |
| **맵 테마** | 배경 테마 변경 | $2.99~4.99 | 네온 시티, 황무지, 눈 내리는 마을 |
| **스킨 번들** | 타워+발사체+승리 이펙트 세트 | $7.99 | 시즌 테마 번들 |

#### 출시 전략
- 초기 출시: 기본 스킨 6종 + 한정 스킨 2종
- 월 1회 신규 스킨 추가 (라이브 운영)
- 시즌/이벤트 한정 스킨으로 희소성 확보

#### 구매 동기
- 개성 표현 (나만의 타워 만들기)
- 수집욕 (전체 스킨 컬렉션 완성)
- 커뮤니티 과시 (리플레이 공유 시)

---

### 3.3 편의 기능 IAP (15% 수익 기여)

**핵심**: 시간 단축, 진행 편의성 제공 (P2W 아님)

#### 편의 기능 상품

| 상품 | 효과 | 가격 | 유형 |
|------|------|------|------|
| **골드 패키지 (소)** | 1,000 골드 | $0.99 | 소모품 |
| **골드 패키지 (중)** | 3,000 골드 | $2.99 | 소모품 |
| **골드 패키지 (대)** | 10,000 골드 | $9.99 | 소모품 |
| **타워 슬롯 확장** | 동시 타워 설치 개수 +2 | $4.99 | 영구 |
| **오토 플레이** | 웨이브 자동 진행 기능 | $2.99 | 영구 |
| **웨이브 스킵 티켓 10개** | 웨이브 즉시 완료 | $1.99 | 소모품 |
| **스타터 패키지** | 골드 2,000 + 타워 슬롯 확장 | $4.99 | 1회 한정 |

#### 골드 밸런스
- 무료 유저: 웨이브당 평균 50~100 골드 획득
- 광고 유저: 광고 활용 시 2배 (100~200 골드)
- 유료 패키지: 진행 막힘 시점에서 1~2시간 절약 효과

---

### 3.4 광고 제거 패스 (10% 수익 기여)

**타겟**: 광고를 싫어하지만 게임을 즐기고 싶은 유저

| 상품 | 효과 | 가격 |
|------|------|------|
| **광고 제거 (영구)** | 리워드 광고 보상 자동 지급 | $2.99 |
| **프리미엄 패스 (30일)** | 광고 제거 + 골드 부스트 +30% | $4.99/월 |

**중요**: 광고 제거 구매 시에도 리워드는 그대로 제공 (광고 시청 없이 보상만)

---

### 3.5 스페셜 타워 IAP (5% 수익 기여)

**핵심**: 밸런스 유지, 무료 타워 대비 ±20% 성능 차이만 허용

#### 스페셜 타워 예시

| 타워 | 특징 | 성능 | 가격 |
|------|------|------|------|
| **레이저 타워** | 관통 공격 (2명 타격) | 기본 타워 대비 -10% DPS | $1.99 |
| **얼음 타워** | 적 속도 -30% 슬로우 | 기본 타워 대비 -20% DPS | $1.99 |
| **독 타워** | DoT 지속 데미지 | 기본 타워와 DPS 동일 | $1.99 |
| **시즌 한정 타워** | 특수 연출 + 고유 이펙트 | 기본 타워와 성능 동일 | $2.99 |

#### 밸런스 원칙
- DPS(초당 피해량)는 기본 타워 ±20% 이내
- 특수 효과는 "전략 다양성" 목적 (성능 우위 X)
- 무료로도 모든 웨이브 클리어 가능하도록 설계

## 4. 재화 시스템

### 4.1 재화 구조

| 재화 | 획득 방법 | 용도 |
|------|----------|------|
| **골드** | 웨이브 클리어, 광고, IAP | 타워 구매/업그레이드 |
| **젬 (프리미엄)** | 첫 클리어 보상, IAP | 스킨 구매, 편의 기능 |

**Simple is Best**: 재화 종류를 2개로 제한하여 혼란 방지

### 4.2 골드 밸런스 (예시)

| 레벨/웨이브 | 무료 유저 골드 | 광고 유저 골드 | 필요 골드 (타워 구매/업그레이드) |
|------------|--------------|--------------|--------------------------------|
| 1~5 웨이브 | 500 | 1,000 | 300 (충분) |
| 6~10 웨이브 | 1,500 | 3,000 | 1,000 (충분) |
| 11~15 웨이브 | 3,000 | 6,000 | 2,500 (적당) |
| 16~20 웨이브 | 5,000 | 10,000 | 6,000 (약간 부족, IAP 유도) |

## 8. LiveOps 수익화 전략

### 8.1 일일 이벤트
- **일일 골드 챌린지**: 특정 조건 달성 시 골드 2배

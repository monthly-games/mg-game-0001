# MG-0001: Tower Defense UI/UX 스펙

> **게임명**: Tower Defense
> **장르**: 타워 디펜스
> **카테고리**: Year 1 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 세로 스크롤 맵 구조
- 상단→하단 적 이동 경로
- 한손 터치로 타워 배치 편의성

---

## 2. Safe Area

| 영역 | 값 | 용도 |
|------|-----|------|
| 상단 | 44-62dp | HUD (웨이브, 골드, HP) |
| 하단 | 34-48dp | 타워 선택 바 |
| 좌우 | 16dp | 맵 여백 |

### 3.1 상단 HUD 요소

| 요소 | 크기 | 위치 | 설명 |
|------|------|------|------|
| 웨이브 표시 | 120×32dp | 좌측 | 현재/총 웨이브 |
| 골드 표시 | 100×32dp | 중앙 | 보유 골드 |
| HP 표시 | 80×32dp | 우측 | 거점 체력 |

### 3.2 타워 선택 바

| 요소 | 크기 | 설명 |
|------|------|------|
| 타워 아이콘 | 56×56dp | 5종 타워 선택 |
| 배속 버튼 | 48×48dp | 1x/2x/3x 속도 |
| 일시정지 | 48×48dp | 게임 일시정지 |

---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 타워 선택 | 56×56dp | 탭 |
| 타워 배치 | 맵 그리드 | 탭 (드래그 앤 드롭 옵션) |
| 타워 업그레이드 | 타워 위 | 탭 → 팝업 |
| 타워 판매 | 팝업 내 | 탭 |
| 맵 스크롤 | 맵 영역 | 스와이프 (맵이 화면보다 클 경우) |

### 터치 피드백
- 타워 선택: 확대 애니메이션 (1.1x, 100ms)
- 배치 가능: 녹색 하이라이트
- 배치 불가: 빨간색 X 표시 + 진동

## 5. 정보 계층

|------|------|----------|

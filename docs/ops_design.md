## 1. 운영 목표

| 항목 | 내용 |
|------|------|
| 포지션 | 공통 엔진 v1의 대표 레퍼런스 타이틀 |
| 역할 | 타워 디펜스+전략 구조의 **튜토리얼 역할** + 안정적인 장기 수익 |
| 타깃 | 캐주얼~미드코어, 신규 유저 유입 창구 |

## 2. 핵심 KPI

| KPI | 목표 | 측정 방법 |
|-----|------|----------|
| 튜토 완료율 | 70% 이상 | 튜토리얼 단계별 이탈 추적 |
| D7 Retention | 25% 이상 | Firebase Analytics |
| 월간 웨이브 보상 수령률 | 80% 이상 | 복귀율/수령 로그 |

| D30 Retention | 12% 이상 | 장기 유지 추적 |

## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 크리티컬 버그 수정 |
| 3~4주 | 튜토리얼/초반 이탈 구간 집중 개선 |
| 5~8주 | 웨이브 보상 밸런스/곡선 튜닝 |
| 9~12주 | 초반 "성장 체감" 이벤트: 스타트 대시, 7일 출석 |

**핵심 과제:**
- 튜토리얼 완료율 70% 달성
- 첫 웨이브 보상 체감 시점 최적화
- 초반 성장 곡선 조정

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 신규 맵/챕터 1차 업데이트 |
| 5개월 | 보스/도전 모드 추가 |
| 6개월 | 타워/적 특성 1종 추가 |
| 7개월 | 신규 맵/챕터 2차 업데이트 |
| 8개월 | 타워/적 특성 2종 추가, 여름 이벤트 |

**핵심 과제:**
- 월 1회 콘텐츠 업데이트 사이클 정착
- 중급 유저 이탈 방지
- 다양한 플레이 스타일 지원

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 시즌제 도입 (Battle Pass v1) |
| 10개월 | 콜라보/크로스 프로모션 1차 |
| 11개월 | 시즌 2 시작, 신규 콘텐츠 |
| 12개월 | 연말 레이드(0012) 연동 이벤트 |

**핵심 과제:**
- 시즌 패스 시스템 안정화
- 크로스 게임 연동 검증
- 1주년 이벤트 준비
